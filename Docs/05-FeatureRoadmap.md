# 功能路线图

## 第一阶段：MVP（最小可行产品）

### 目标
验证核心体验：用户能浏览和创建前端作品

### 功能清单

#### 认证模块
- [ ] Apple 登录
- [ ] Google 登录
- [ ] 登出
- [ ] 用户会话管理

#### 作品浏览
- [ ] 作品列表（推荐流）
- [ ] 作品详情（WebView 渲染）
- [ ] 上下切换作品（按钮）
- [ ] 作品基本信息展示

#### 作品互动
- [ ] 点赞/取消点赞
- [ ] 收藏/取消收藏
- [ ] 分享作品

#### 个人中心
- [ ] 查看个人信息
- [ ] 查看自己的作品
- [ ] 查看收藏的作品

#### 作品创作（简化版）
- [ ] 基础代码编辑器
- [ ] HTML/CSS/JS 三个输入区域
- [ ] 实时预览
- [ ] 发布作品
